Tentang Aku

Proyek Unggulan

ICU Simulator

Simulasi medis interaktif menggunakan Arduino. Antarmuka diprogram dengan pustaka Adafruit_SH110X untuk layar OLED via I2C (SDA 21, SCL 20).

Server Installer

Skrip Bash otomatis untuk instalasi dan konfigurasi server Minecraft (Java/Bedrock), mendukung integrasi plugin Geyser dan Floodgate.

Radio Transceiver

Eksperimen komunikasi jarak jauh menggunakan modul nRF24L01+ dan mikrokontroler untuk pengiriman paket data nirkabel.

Triple-Boot System

Konfigurasi partisi dan bootloader manual untuk menjalankan Windows, Arch Linux, dan Android TV secara berdampingan.
